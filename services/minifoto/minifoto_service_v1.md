# Обзор бесплатных сервисов для создания миниатюр (минифото)

> Версия 1 · Составлена для проекта [family-tree](https://github.com/bpmbpm/family-tree)

## Задача

Создать миниатюру портретного фото (≈ 76×74 пкс, PNG, RGB) из большого семейного снимка:
выбрать лицо рамкой захвата, обрезать, отмасштабировать и сохранить в папку `ver2/pic/`.

---

## Собственный инструмент проекта

| Инструмент | Файл | Описание |
|---|---|---|
| **Minifoto v1** | `services/minifoto/minifoto_v1.html` | Браузерный HTML/JS. Загрузка фото, подвижная рамка захвата, редактор (масштаб, сдвиг, яркость, контраст), предпросмотр, сохранение PNG. Работает локально без сервера. |

---

## Сравнение бесплатных онлайн-сервисов

### 1. [Crop & Resize — Squoosh](https://squoosh.app/)

| Параметр | Значение |
|---|---|
| Разработчик | Google (открытый исходный код) |
| Цена | Бесплатно |
| Работа офлайн | Да (PWA) |
| Обрезка | Да (свободная, с соотношением) |
| Изменение размера | Да (в пикселях или процентах) |
| Форматы выхода | PNG, JPEG, WebP, AVIF и др. |
| Яркость/контраст | Нет |
| Пакетная обработка | Нет |
| Конфиденциальность | Высокая: файлы не покидают браузер |
| **Итог** | Отлично для разовой обрезки + конвертации; нет подвижной рамки «поверх» фото |

---

### 2. [ILoveIMG — Crop Image](https://www.iloveimg.com/crop-image)

| Параметр | Значение |
|---|---|
| Цена | Бесплатно (с ограничениями); платный план |
| Работа офлайн | Нет |
| Обрезка | Да (визуальный редактор, пропорции) |
| Изменение размера | Да |
| Форматы выхода | JPG, PNG, GIF и др. |
| Яркость/контраст | Нет |
| Пакетная обработка | Да (до 30 файлов в бесплатном плане) |
| Конфиденциальность | Файлы загружаются на сервер |
| **Итог** | Удобен для пакетной обрезки; требует соединения; данные покидают устройство |

---

### 3. [Photopea](https://www.photopea.com/)

| Работа офлайн | Частично (кэш браузера) |
| Обрезка | Да (инструмент кадрирования, заданный размер) |
| Изменение размера | Да (пикселей/дюйм/см) |
| Форматы выхода | PNG, JPEG, PSD, PDF и мн. др. |
| Яркость/контраст | Да (слои, коррекция, кривые) |
| Пакетная обработка | Нет |
| Конфиденциальность | Высокая: файлы не покидают браузер |
| **Итог** | Полноценный фоторедактор онлайн; избыточен для простой обрезки миниатюр |

### 6. [GIMP](https://www.gimp.org/) (десктоп, бесплатно)

| Параметр | Значение |
|---|---|
| Цена | Бесплатно, с открытым исходным кодом |

| Обрезка | Да (инструмент кадрирования с заданным размером) |

| **Итог** | Максимальная функциональность; высокий порог вхождения для новых пользователей |

## Вывод

Для задач проекта **family-tree** наиболее подходящий подход — использовать встроенный инструмент **`minifoto_v1.html`**:
- полностью работает в браузере без сервера,
- все фотографии остаются на устройстве пользователя,
- предустановленный целевой размер 76×74 пкс (оптимальный на основе анализа `ver2/pic/`),
- не требует установки программ или регистрации.

Из сторонних инструментов, **Squoosh** рекомендуется как наилучшая бесплатная альтернатива: открытый код от Google, полностью локальная обработка, поддержка PNG.
