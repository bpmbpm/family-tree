## I. МЕТОДЫ ИССЛЕДОВАНИЯ

### 1. Улучшение качества исходных данных

**Проблема:** В текущем кейсе (Кононов А.Н.) выявлено критическое противоречие топонимов — село Елпатьево существует в Ярославской области, а не в Калязинском районе Тверской области.

**Рекомендации:**
- Добавить в шаблон блока 3 обязательное поле **«Губерния/уезд до 1917»** для каждого топонима
- Включить проверку топонимов через справочники (etomesto.ru, rodnaya-zemlya.ru) на этапе заполнения исходных данных
- Требовать указание **источника информации** для каждого факта (документ/семейная легенда/фотография)

### 2. Приоритизация поиска

**Проблема:** Широкий охват источников в промпте может приводить к неэффективному расходу времени.

**Рекомендации:**
- Ввести **уровни приоритета** поиска:
  - **P1 (критический):** Прямые военные базы (pamyat-naroda, obd-memorial) для участников ВОВ
  - **P2 (высокий):** Региональные архивы по месту жительства
  - **P3 (средний):** Общие генеалогические базы (FamilySearch, Яндекс.Архив)
  - **P4 (низкий):** Форумы, соцсети, пользовательские деревья

### 3. Автоматизация нормализации

**Проблема:** Ручное составление вариантов ФИО и топонимов занимает значительное время.

**Рекомендации:**
- Создать базу данных типичных замен букв для OCR-ошибок
- Разработать инструмент автоматической генерации вариантов написания фамилий
- Использовать справочники транслитерации (ISO, BGN/PCGN) для латинских форм

### 4. Структурирование семейных связей

**Рекомендация:** Использовать стандартный формат GEDCOM для представления семейных связей, что позволит:
- Автоматически визуализировать родословное древо
- Импортировать данные в специализированные программы
- Обмениваться данными между исследователями

## V. ЗАМЕЧАНИЯ ПО ТЕКУЩЕМУ КЕЙСУ (КОНОНОВ А.Н.)

### Выявленные проблемы

1. **Противоречие топонимов:** Село Елпатьево в исходных данных указано как часть Калязинского района Тверской области, однако село Елпатьево существует в Переславском районе Ярославской области. Требуется уточнение.

2. **Неполнота данных:** Отсутствует год рождения исследуемого лица, что критически затрудняет поиск в военных базах.

3. **Отсутствие документов:** Нет ни одного первичного документа (паспорт, военный билет, свидетельства), что снижает надёжность всех построений до уровня гипотез.

### Рекомендации по данному кейсу

1. Уточнить у заказчика точное местоположение Елпатьево
2. Запросить копии свидетельств о смерти детей/жены для извлечения дополнительных данных
3. Проверить похозяйственные книги 1930-х годов в районном архиве
