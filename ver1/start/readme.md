## minimum
Строим генеалогическое дерево из файла excel. Напиши программу на Браузерном js, весь код в index.html.  
Рендерим схему в mermaid. При наличии специальной библиотеки mermaid для генеалогического дерева используем ее. Дерево должно быть похоже на www.myheritage.com
При старте index.html данные считываются из tree.xlsx с листа person. Идентификаторами в mermaid являются idA, на схеме отображаются поля: label, birth, death. Дерево строится на основе предикатов hasFather и hasMother.
Внизу схемы должна быть кнопка «mermaid.live», которая формирует ссылку и передает в https://mermaid.live/edit с использованием pako (https://mermaid.live/edit#pako).
В папке .pic хранятся минифото для персонажей дерева в одноименном файле с idA и расширением .png. При отсутствии файла в качестве минифото подставляется dafaultm.png для мужчин (male, М,  см. поле sex) или dafaultf.png для женщин (female, Ж,  см. поле sex).
